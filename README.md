# 🧠 LeetCode Practice Repository

This repository contains my solutions to various coding challenges from LeetCode.  
The goal is to improve problem-solving skills, algorithmic thinking, and consistency through practice.

---

## 📁 Structure

Each problem is organized in its own **numbered folder**, following this format:

```
/001-two-sum
/002-add-two-numbers
/003-longest-substring-without-repeating-characters
...
```

### Folder Naming Convention

- Starts with a **zero-padded number**
- Followed by the **problem name in kebab-case**

Example:

```
/015-3sum
/121-best-time-to-buy-and-sell-stock
```

---

## 📌 Contents of Each Folder

Each folder may include:

- ✅ Problem description (or link to LeetCode)
- 💻 Solution implementation
- 🧾 Explanation (optional but recommended)
- 🧪 Test cases (if applicable)

Example:

```
001-two-sum/
├── solution.java
├── README.md
```
